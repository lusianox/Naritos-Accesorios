## Run Locally

Para instalar la pagina de Naritos Accesorios, solo tenes que clonar el repositorio.

```bash
  git clone https://github.com/lusianox/Naritos-Accesorios.git
```

Y despues correr el "Live Server" en Visual Studio Code.
